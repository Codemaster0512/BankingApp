README - Smart Banking System (JavaFX)

Project Name:
Smart Banking System

Description:
This is a JavaFX based mini banking project with 3 pages:

1. Welcome Page
2. Login Page
3. Banking Dashboard

Features:
✔ Attractive UI Design
✔ Keyboard Shortcuts
✔ Deposit Money
✔ Withdraw Money
✔ Balance Display
✔ Multi-page Navigation

KEYBOARD SHORTCUTS
----------------------------------------

ENTER Key:
- Welcome Page → Open Login Page
- Login Page → Login
- Dashboard → Deposit Amount

ESC Key:
- Login Page → Back to Welcome Page
- Dashboard → Back to Login Page

----------------------------------------
DASHBOARD FUNCTIONS
----------------------------------------

1. Deposit:
Enter amount and press Deposit button or ENTER key.

2. Withdraw:
Enter amount and click Withdraw button.

3. Exit:
Returns to Welcome Page.
